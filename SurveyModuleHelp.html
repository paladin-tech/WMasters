For Auger, Geotech and other wells 
If well is completed and there is a refusal, that will go in column A, another refusal in column B and so on. Once a final well is drilled that will go in the base. 
If a mistake is made just send us the coordinates and we will change them. 
  
For Resource wells 
You always insert info in the BASE because we are assuming that they are as-built coordinates 
You will have two possible scenarios: 
FIRST: if you have a note from the Drilling Team that you need to as-built them again then you will send us this information in the attached format and we will change them – you will fill up the base coordinates for this scenario – and we will override the existing base. In this scenario we still should capture the original coordinate so maybe move original that was in base (prestake) over to A and refill base with actual asbuilt. 
SECOND: If you have a Re-drill you will send us that info and you will insert those coordinates under A – you will have to send that to us because at this time you already have inserted that into the base so the  A column is locked and only we can insert them.  That is fine because we are not going to be doing that many re-drills anyway.
